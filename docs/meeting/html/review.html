Review Module

Base Path: /api/reviews
Review APIs to submit ratings and manage moderation views.

Endpoints

MethodPathRoleDescription
POST/USERSubmit review and rating for equipment.
GET/equipment/{id}PUBLICGet all reviews for one equipment.
GET/ADMINGet all reviews sorted by createdAt desc.
DELETE/{id}ADMINDelete inappropriate review.

Request Samples

Submit review

Endpoint: POST /api/reviews/

{
  "userId": 2,
  "equipmentId": 1,
  "rating": 5,
  "comment": "Excellent condition and smooth pickup experience."
}

Flow (Service-Level)

  1. Service validates user and equipment existence.
  2. Prevents duplicate review per user+equipment pair.
  3. Persists review with rating (1-5) and optional comment.
  4. Equipment reviews are returned newest-first for display.
  5. Admin can remove a review by id when moderation is needed.

Error Handling

Validation and business exceptions are returned in a consistent error format from GlobalExceptionHandler with fields: timestamp, status, error, message, path, and validationErrors.
