Admin Module

Base Path: /api/admin
Administrative APIs for user management and dashboard-level operational metrics.

Endpoints

MethodPathRoleDescription
GET/usersADMINList all users sorted by createdAt desc.

Deactivate user

Flow (Service-Level)

  1. Controller delegates to AdminService.
  2. AdminService aggregates counts from UserRepository, RentalRepository, and PaymentRepository.
  3. Deactivate operation marks the user as deleted without physical delete.
  4. Dashboard revenue uses payment status based sums (SUCCESS and PENDING).

Error Handling

Validation and business exceptions are returned in a consistent error format from GlobalExceptionHandler with fields: timestamp, status, error, message, path, and validationErrors.
