Notification Module

Base Path: /api/notifications
Email notification API for payment success confirmation using configured SMTP.

Endpoints

MethodPathRoleDescription
POST/USERSend payment success email for a rental with SUCCESS status.

Request Samples

Send payment success notification

Endpoint: POST /api/notifications/

{
  "rentalId": 1
}

Flow (Service-Level)

  1. Fetch payment by rentalId.
  2. Reject request unless payment status is SUCCESS.
  3. If notification.mail.enabled=false, email send is skipped safely.
  4. If enabled, JavaMailSender sends message to payer email.
  5. Returns delivery metadata in NotificationResponse.

Error Handling

Validation and business exceptions are returned in a consistent error format from GlobalExceptionHandler with fields: timestamp, status, error, message, path, and validationErrors.
